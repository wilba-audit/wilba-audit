# GEO (Generative Engine Optimization) — LLM Discoverability Hack

> Tip from a specialist who's done this across multiple pages with consistent results.

## The Process

1. **Search** for the topic you want to be found for
2. **Look at the AI result** (in any LLM or Google AI search)
3. **Copy the text** — specifically the header titles and related keywords
4. **Build a web page matching the above** — same copy structure as what the search result says, with headers matching the related keywords
5. **Each text block** (per related keyword): **320–420 characters**
6. **Total page text**: **800–1,200 words**

## Why It Works

LLMs pull from pages that mirror the structure and language they already use in their responses. By matching the AI's own output format — headers, keyword density, block length — you're essentially training the model to cite your page as a source.

## Use Cases for WILBA

- wilba.ai landing pages optimized for "AI receptionist for service businesses"
- Audit landing page optimized for "AI automation audit for small business"
- Blog/content pages targeting specific AI search queries
