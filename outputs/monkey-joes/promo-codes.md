## Promo Code Tracking — How We Use This

Every code is unique to a channel. Each week, we export Aloovy code redemption data:

- `WACKY-POL` redemptions → came from Pointe Orlando paid ads
- `WACKY-WP` redemptions → came from Winter Park paid ads
- `WELCOME` redemptions → came from email opt-in campaign
- `COMEBACK` redemptions → came from dormant re-engagement

This tells us exactly which channel is delivering paying customers — and where to increase (or cut) spend.
