## Notes on Pulling the Data

### Facebook Ads Manager
1. Go to Ads Manager → select date range (Mon–Sun prior week)
2. Columns to pull: Spend, Link Clicks, Conversions (BOGO purchase), Cost per Result
3. Filter by: Pointe Orlando account / Winter Park account separately

### Constant Contact
1. Dashboard → Campaigns → filter by "last 7 days"
2. Click each campaign: copy Open Rate, Click Rate, Total Sent
3. Automations: Reports → Automation → check each active sequence

### Aloovy
1. Log in → Reports → Promo Code Usage
2. Filter by date range
3. Export or screenshot each code's redemption count
4. Check Frequent Jumper Card product sales separately

### CallRail (once live)
1. Dashboard → Calls → filter by date range
2. Attribution → which campaign / source drove each call
