# The WILBA Content Engine

> Your organic content weapon for Instagram + YouTube.
> Fuel: The Rundown AI (daily) · Voice: your Brand Bible · Studio: Higgsfield · Face + strategy: Jess.

News in one end → finished, on-brand content out the other. You stay the face and the director; the blank page disappears.

---

## The buttons you press (commands)

| Command | What you get |
|---|---|
| **`/content`** | The full daily loop. Rundown → angles → a ready-to-shoot reel + visuals → daily brief. **Start here each morning.** |
| **`/rundown`** | Just today's AI news translated into 2-3 WILBA angles. 60-second scan. |
| **`/make-reel [topic]`** | Full reel package (hooks + script + caption + hashtags) + visuals. |
| **`/make-carousel [topic]`** | A branded carousel, slides written and designed. |
| **`/thumbnail [title]`** | 3-4 YouTube thumbnail options to choose from. |
| **`/content-week`** | A whole week planned + written in one batch. The leverage move. |
| **`/repurpose [video/script]`** | One YouTube video → Reel, Short, carousel, Story, email. |
| **`/post-check [draft]`** | Brand + virality gut-check before you post. |

## The workers (agents — run automatically by the commands)

| Agent | Role |
|---|---|
| **trend-scout** | Reads The Rundown AI, surfaces the angles that matter |
| **hook-writer** | Turns angles into copy in your exact voice |
| **visual-producer** | Runs Higgsfield to make thumbnails, carousels, b-roll, faceless reels + scores virality |
| **repurposer** | Turns one piece of content into six |

## The source of truth these all read
- `outputs/brand/brand-positioning.md` — voice, 15 rules, banned phrases, visual palette
- `outputs/brand/social-media-intelligence.md` — 4 segments, 4 pillars, 20 hooks, 30-day calendar

## Where things get saved
- Daily briefs → `outputs/content/YYYY-MM-DD-daily-brief.md`
- Reels → `outputs/content/reels/`
- Carousels → `outputs/content/carousels/`
- Thumbnails → `outputs/content/thumbnails/`
- Weekly plans → `outputs/content/weeks/`
- Running logs → `rundown-log.md`, `visual-log.md`

## Connections powering this
- **Gmail** — The Rundown AI arrives from `news@daily.therundown.ai`
- **Higgsfield** — Plus plan, your visual studio (images, video, thumbnails, virality prediction)
