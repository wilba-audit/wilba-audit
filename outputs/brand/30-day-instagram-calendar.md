## Jess Morrell / WILBA — Copy-Paste Ready

**Period:** Start any Monday — this is a rolling 30-day system
**Platforms:** Instagram (primary) — Reels, Carousels, Stories
**CTA default:** "Free audit — link in bio" unless otherwise specified
**Optimal posting times (AEST):** Reels + Carousels → 7:00–8:30am or 7:00–8:30pm | Stories → any time, multiple per day

**How to use this calendar:**
1. The hook, concept, and CTA are written and ready — copy them directly into your script or caption
2. Stories run daily regardless of main feed post — use the daily Story prompts
3. Film Reels in batches (Monday mornings work well for the week ahead)
4. Adjust posting times by ±30 minutes based on your own analytics once you have 2 weeks of data
