# /make-carousel [topic] — A Branded Carousel, Slides Designed

> The highest-save-rate format in your niche. Give it a topic; get slides ready to post.

## Instructions

**Topic:** `$ARGUMENTS` — if empty, use today's #1 Rundown angle (launch **trend-scout** first) or the next carousel slot in the 30-day calendar.

### Step 1 — Write the slides
Launch the **hook-writer** agent for a Carousel. Get back:
- COVER slide (works as a standalone hook / reel cover)
- 5-8 slides, slide-by-slide (headline + supporting line)
- FINAL slide = the CTA (free audit)
- A short caption

Pick the strongest structure for the topic: listicle / before-after / myth-bust / formula-calculation / case-study.

### Step 2 — Design the slides
Launch the **visual-producer** agent to generate the carousel graphics in Higgsfield — 4:5 or 1:1, brand palette (navy + sand + ocean-blue accent), clean geometric sans, one thought per slide, legible on mobile. Generate the cover as the hero, then supporting slides.

### Step 3 — Deliver
Save to `outputs/content/carousels/YYYY-MM-DD-[slug].md` with the slide text + generated graphics + caption. Chat summary: the cover hook + "post-ready, [N] slides attached."

Check against Brand Bible Section 5d (text/typography) and 5g (anti-guidelines). No all-caps body text, two font weights max, never off-palette.
