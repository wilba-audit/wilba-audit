# /make-reel [topic] — A Full Reel Package, Ready to Shoot

> Give it a topic (or nothing, and it'll use today's #1 Rundown angle). Get back everything you need to record and post.

## Instructions

**Topic:** `$ARGUMENTS` — if empty, launch **trend-scout** first and use the #1 pick of the day.

### Step 1 — Write it
Launch the **hook-writer** agent for a Reel on the topic. Get back:
- 3 hook options (first 2-3 seconds) with a recommended pick
- The 30-60 sec script in Jess's voice
- On-screen text cards
- The caption (hook → who it's for → value → one CTA)
- 5-10 hashtags

### Step 2 — Make the visuals
Launch the **visual-producer** agent to create the matching visual — a thumbnail/cover frame, or b-roll / faceless reel video in 9:16, on-brand palette. Preflight credits.

### Step 3 — Deliver
Save to `outputs/content/reels/YYYY-MM-DD-[slug].md` with the copy + visuals + a 1-line shoot note (setting, wardrobe, framing from Brand Bible Section 5). Give Jess a 3-line chat summary: the hook, what's attached, and "record this."

Everything checks against `outputs/brand/brand-positioning.md`. Run the banned-phrases check before delivering.
