# Data-Collection Checklist

Fill this before finalising an audit. Live analytics beat guesses every time.
Anything you cannot pull, mark `[CLIENT TO CONFIRM]` in the audit — never invent
a number. Screenshots of native analytics are ideal.

## Access / handles (per platform the client uses)
- [ ] Handle / URL
- [ ] Account type (creator/business/personal — affects analytics access)
- [ ] Can we get analytics access or dated screenshots? (Y/N)

## Per-platform metrics — **last 90 days**
For **Instagram / TikTok / Reels / Shorts:**
- [ ] Follower count + 90-day net change
- [ ] Posts published (count) and cadence
- [ ] Reach — total, and % to non-followers
- [ ] Average view duration / % watched (retention)
- [ ] Saves, shares/sends, comments (totals + per-post average)
- [ ] Profile visits and follows-from-reach %
- [ ] Link clicks (bio / stories)
- [ ] **Top 3 posts** (link + why they worked)
- [ ] **Bottom 3 posts** (link + why they flopped)

For **YouTube long-form:**
- [ ] Subscribers + 90-day change
- [ ] Videos published
- [ ] Impressions, click-through rate (CTR), average view duration + %
- [ ] Traffic sources (search vs suggested vs external)
- [ ] Top/bottom videos

For **LinkedIn / X:**
- [ ] Followers/connections + change
- [ ] Post impressions, engagement rate, comment quality
- [ ] Profile views / search appearances

## Business context (the part that makes it strategic, not cosmetic)
- [ ] The real offer and its price point / margin
- [ ] Current source of clients (so we know social's actual job)
- [ ] Enquiries currently attributable to social (even roughly)
- [ ] The ICP: one specific person — age, situation, the 2am worry, the trigger to act
- [ ] What "success" looks like to the client in 6 months (in their words)
- [ ] Total spent to date on content + who produced it
- [ ] Any compliance constraints (for regulated niches: GMC/ASA/legal)
- [ ] Assets on hand (existing footage, email list size, website traffic)

## Competitive / benchmark scan
- [ ] 3–5 accounts in the niche who ARE getting traction (what they do differently)
- [ ] The client's category benchmark for engagement/retention (rough)

## Qualitative review (do this yourself)
- [ ] Watch 10 recent posts. Note hook quality, ICP fit, value density, CTA.
- [ ] Read the bio/profile as a cold ICP visitor: is it instantly clear?
- [ ] Trace the conversion path end-to-end. Where does it dead-end?
